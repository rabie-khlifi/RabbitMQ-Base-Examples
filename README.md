# RabbitMQ Basics

This project contains small RabbitMQ examples grouped under two folders:

- [examples-js](examples-js): JavaScript versions of the examples.
- [examples-ts](examples-ts): TypeScript versions of the same examples.

## Install

1. Make sure RabbitMQ is running locally on `amqp://localhost`.
2. Install dependencies:

```bash
npm install
```
